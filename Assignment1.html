Q1. What is Web 1.0, What are the uses of web 1.0?

Sol:

Web 1.0 is the ready only web. In other words ,the early web allowed us to search
for information and read it.

Uses of Web 1.0: Web 1.0 functions as CDN(content delivery network),
allowing a chunk of the websites to be displayed on the websites.
As a result, it can be used as personal website. The users
would be charged in terms of each page view.

Q2. What is Web 2.0, What are the uses of web 2.0?

Sol:

Web 2.0 are websites and applications that make use of user generated content for end users.

Uses of Web 2.0: With web 2.0 , Websites enable community used input,
interaction, content sharing and collaboration. Types of social media sites
and applications include forums, microblogging, social networking, social bookmarking,
social cution and wikis.

Q3. What is Web 3.0, What are the uses of web 3.0?

Sol:

Web 3.0 is the concept of the next genaration of the web, in which most users will be connected via
a decentralized network and have access to their own data

Uses of Web 3.0: With blockchain at its foundation,
Web 3.0 will enable new applications and services based on blockchain technologies
in active use to day. They include the following: NTFs(Non-fungible tokens.
NTFs are a type of unique cryptographic asset used to create and
authenticate ownership of digital assets.
